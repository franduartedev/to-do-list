# To-Do List

Una aplicación web simple para agregar y borrar tareas, hecha con **HTML, CSS y JavaScript puro**.

---

## Funcionalidades
- Agregar tareas escribiendo en el input.
- Limpiar todas las tareas con el botón "Borrar tareas".
- Agregar tareas presionando la tecla **Enter**.
- Validación para evitar tareas vacías.

---

## Tecnologías
- HTML
- CSS
- JavaScript
